## Common Measure Names

- `Revenue` / `Net Revenue` / `Gross Revenue`
- `Cost of Goods Sold (COGS)`
- `Gross Profit` = Revenue − COGS
- `Operating Expenses (OpEx)`
- `EBITDA` = Earnings Before Interest, Tax, Depreciation, Amortisation
- `EBIT` = EBITDA − Depreciation − Amortisation
- `Net Profit` / `Net Income`
- `Budget` / `Forecast` / `Actuals`
- `Variance` = Actuals − Budget
- `Variance %` = Variance / Budget

## Common Table Names

- `FactFinancials` / `Fact_GL`
- `DimDate` / `Date`
- `DimAccount` / `ChartOfAccounts`
- `DimCostCentre` / `DimDepartment`
- `DimEntity` / `DimCompany`
- `BudgetFact` / `FactBudget`

## Account Types

- `P&L` — Profit & Loss (income statement)
- `BS` — Balance Sheet
- `CF` — Cash Flow
